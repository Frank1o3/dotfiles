# What is this?

This repo contains a config for hyprland that will setup themes a them selector via mod + w or in better terms super + w and other things like full deskop themes based of your wallpaper

# Goals

1. Refine the waybar to look cleaner
2. Make nice transition animations for the themes
3. Have full support for OS keyring's and a nice dunst config
4. Make the hotkeys less confusing and more universald

# How to install it

You can run one of the following commands:
```bash
curl -fsSL https://raw.githubusercontent.com/Frank1o3/dotfiles/main/install.sh | bash

curl -fsSL https://raw.githubusercontent.com/Frank1o3/dotfiles/main/install.sh | zsh

curl -fsSL https://raw.githubusercontent.com/Frank1o3/dotfiles/main/install.sh | fish
```
